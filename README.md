## Host On A Subdomain

Sample setup for GitHub Pages:

1. Pick a subdomain, for example `portfolio.example.com`.
2. In your domain DNS, create a `CNAME` record:
	- Name/Host: `portfolio`
	- Target/Value: `<your-github-username>.github.io`
3. Add a file named `CNAME` in the repository root with this exact line:
	- `portfolio.example.com`
4. Push the repo to GitHub Pages and enable Pages in the repository settings.
5. Wait for DNS to propagate, then open the subdomain in your browser.
